## Folder Structure

### `/archive`
Historical documentation organized by purpose:
- **`/implementation`** - Implementation notes and step-by-step guides
- **`/setup`** - Original setup docs (consolidated into SETUP.md)
- **`/reviews`** - Code reviews, audits, and analysis
- **`/guides`** - Feature guides, quick references, troubleshooting
- **`/status`** - Completion reports, summaries, and milestones

See [`archive/README.md`](./archive/README.md) for detailed archive organization.

### `/setup`
Setup-related subdirectory (non-consolidated docs):

- Monetization and pricing
- Product requirements
- Executive summaries

### `/roadmap`
Product roadmap and feature planning:
- Cricut/Etsy integration plans
- Future enhancements
- Development notes
